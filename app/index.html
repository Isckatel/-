Приложение разработано на Vue CLI

Автоматизированная система управления технологическим процессом (АСУ ТП) — группа решений технических и программных средств, предназначенных для автоматизации управления технологическим оборудованием на промышленных предприятиях.

Я писал программы для программируемых логически контроллеров в среде разработке CODESYS. ПЛК программируются на языках, определённых стандартом МЭК61131. В основном мы использовали ST (Structured Text) — текстовый паскалеподобный язык.

За время работы мной были реализованы следующие проекты:

  • 2 варианта ПО для контроллера материальных шлюзов;
  • 2 варианта ПО для контроллера управления насосами;
  • ПО на деаэратор;
  • ПО для контроллера трехимульсного регулятор уровня воды барботёра парового котла;
  • Пользовательский интерфейс для ПО одногорелочного воздухонагревателя;
  • ПО для контроллера емкости хранения и дозирования эмульсии;
  • ПО на панель оператора управления материальным шлюзом.
